HTML Elemente

Grundgerüst

HTML bildet das Grundgerüst jeder Webseite. Ausgesprochen bedeutet HTML "Hyper Text Markup Language" oder auf Deutsch "Hypertext-Auszeichnungssprache". Das bedeutet, dass es eine Beschreibende Sprache ist. Es wird also mit den einzelnen Elmenten beschrieben, was man darstellen möchte.

Zusätzlich zu den HTML Elementen wird auch ein CSS Stylesheet geladen. Dieses definiert, wie die einzelnen Elemente formatiert werden sollen.

Interpretation

Bei HTML gibt es keine Prozesslogik wie bei anderen Programmiersprachen. Der Browser interpretiert beim Empfang der Webseite den HTML Code und generiert die einzelnen Elemente. Dabei wird das Stylesheet geladen, welches bestimmt, wie die Elemente aussehen müssen und rendert die Seite entsprechend im Browsertab.

HTML Elemente

Eine Webseite ist immer aus unterschiedlichen HTML Elementen aufgebaut. Je nachdem, was man gerne haben möchte gibt es bestimmte Elemente, welche man verwenden kann. Grundsätzlich kann man jedoch mit CSS jedes Element so manipulieren, wie man gerne möchte. Warum soll man sich also die Mühe machen? Die Gliederung des Inhalts in die dazu passenden Elemente hilft bei Menschen mit Sehbehinderung auf der Webseiten mit ihrem Sehhilfeprogramm zu navigieren. Diese Orientiern sich an den unterschiedlichen HTML Elementen. Werden die Elemente nicht wie eigentlich gedacht verwendet, erschwert das die Navigationsfähigkeit für diese Personen.

Übersicht

Element Zweck
html Das html Element ist das äusserste Element einer Webseite und beinhaltet alle anderen Elemente
head Das head Element ist nicht Sichtbar. Darin werden Metainformationen gespeichert wie Sprache, Stylesheets, Schriftarten, Seitentitel usw.
body Das Body Element beinhaltet alle Webseitenelemente und ist innerhalb des html Elements.
header Das headerelement wird zum Erstellen des Kopfteils verwendet und beinhaltet meist auch das Navigationsmenü
body Im body Element platziert man den Seiteninhalt, wie Texte Bilder oder was auch immer man auf der Webseite abbilden möchte
footer Das footer Element wird für den Abschliessenden Teil einer Webseite verwendet. Viele Webseiten platzieren dort ihren Copyright Banner und oft auch nochmals ein simplifiziertes Manü
article Im article Element werden meistens Texte eingebettet, welche die einzelnen Artikel einer Webseite dartstellen.
div Das div Element ist für die Diversen Ansprüche, welche man nicht mit einem der oberen Elemente erreichen kann. Oft wird auf Webseiten auch das div Element anstatt des article Elements verwendet.
input Mit dem Inputelement lassen sich diverse Eingabemasken erstellen. Welche Möglichkeiten du hast findest du hier.
ul ul ist die Abkürzung für "unsorted list". Diese Liste erzeugt für jedes Element Aufzählungszeichen wie Punkte, Kreise, Striche usw.
ol ol ist die Abkürzung für "ordered list". Anstatt Aufzählungszeichen besitzt diese Listenform Zahlen.
img Das img Element wird für Bilder verwendet. Alternativ gibt es auch video oder audio für die unterschiedlichen Medientypen
h1, h2, h3, h4, h5 Die h Elemente werden für Titel verwendet. Je kleiner die Zahl nach dem "H" umso grösser die Schrift. Es bezeichnet somit die Abfolge der Titel/Untertitel. h1 = Haupttitel, h2 = Untertitel von h1, h3 = untertitel von h2 usw.
